## 桥接模式

### 前言

桥接模式的代码实现非常简单，但是理解起来稍微有点难度，并且应用场景也比较局限，所以，相当于代理模式来说，桥接模式在实际的项目中并没有那么常用，所以能分辨出来就行了，这里不做重点的学习。   

### 定义

桥接模式：将抽象部分和它的实现部分分离，使他们都可以独立的变化。
